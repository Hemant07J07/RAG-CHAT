# RAG-CHAT

RAG-CHAT is a full-stack retrieval-augmented chatbot project.

## Structure

- `backend/`: FastAPI API for retrieval, web search, and Ollama chat
- `frontend/`: Next.js chat interface with conversation history

## Run

### Backend

1. `cd backend`
2. `uvicorn app:app --reload --port 8000`

### Frontend

1. `cd frontend`
2. `npm run dev`
